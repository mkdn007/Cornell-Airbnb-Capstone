# Cornell-Airbnb-Capstone
This project builds a hybrid analytics engine for NYC Airbnb hosts that combines a log-linear pricing regression with KNN distance metrics for competitive cohort benchmarking to pinpoint exact nightly revenue losses and prescribe specific operational and amenity upgrades.

The dataset was obtained from Inside Airbnb, an independent, non-commercial open-source data project that aggregates publicly available data directly from the Airbnb platform. The specific data snapshot is for New York City captured on June 14, 2026.
